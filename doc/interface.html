init function

Инициализация библиотеки. 

Syntax

C
int __cdecl init(
 __in const char* config
);

Parameters

config [in]

Type: char*

Параметры инициализации библиотеки в формате JSON.

Return value

Type:

Type: int

Если инициализация проходит успешно возвращается значение, отличное от нуля, иначе 0.

Remarks

Парамеры запуска берутся  из  config:

downloadDir - директория для сохранения файлов.

httpPorts - диапазон портов для вэб-сервера.

Пример

{
  "downloadDir": "c:\downloads",
  "httpPorts": {
    "min": 100,
    "max": 200
  }
}

add_torrent function

Добавление торрента по .torrent файлу.

Syntax

C
int __cdecl add_torrent(
 __in const char* path
);

Parameters

path [in]

Type: char*

Путь к .torrent файлу.

Return value

Type:

Type: int

Если функция завершается успешно, возвращается ID добавленного торрента, иначе -1.

Remarks

Закачка торрента начинается только после вызова start_download. 

При каждом вызове данной функции создается новый каталог, в который будут сохраняться все файлы торрента. Имя каталога формируется из downloadDir + случайное уникальное имя.

Например "downloadDir\abcd123"

get_torrent_files function

Получение информации об определенном торренте.

Syntax

C
const char* __cdecl get_torrent_files(
 __in int torrent,
);

Parameters

torrent [in]

Type: int

ID торрента.

Return value

Type:

Type: char*

Возвращает информацию об определенном торренте в формате JSON .

Remarks

Результатом выполнения функции является массив, содержащий id, размер, путь каждого файла.

 

Пример:

[
  {
    "size": 123,
    "path": "c:\downloads\abcd123\file1.avi",
    "id": 0
  },
  {
    "size": 345,
    "path": "c:\downloads\abcd123\file2222.avi",
    "id": 1
  }
] 

Важно:  Указатель, возвращаемый функцией, должен быть актуален все время, вплоть до вызова delete_torrent.

start_download function

Старт закачки определенного файла торрента.

Syntax

C
char* __cdecl start_download(
 __in int torrent,
 __in int file_id
);

Parameters

torrent [in]

Type: torrent_id

ID загрузки.

file_id [in]

Type: int

ID файла.

Return value

Type:

Type: char*

Возвращает урл для скачки данного файла по http, либо пустую строку в случае неудачи.

Remarks

Стартует последовательная закачка указанного в параметрах функции файла с нулевой позиции. 

Генерируется урл для отдачи данного файла по http.

Важно:  Указатель, возвращаемый функцией, должен быть актуален все время, вплоть до вызова stop_download.

delete_torrent function

Остановка закачки и удаление торрента.

Syntax

C
void __cdecl delete_torrent(
 __in int torrent
);

Parameters

torrent [in]

Type: int

ID загрузки.

Return value

Type:

Type: void

Нет возвращаемых значений.


Remarks

Останавливается закачка всех файлов данного торрента и сам торрент удаляется из торрент-клиента.

Каталог, который был создан в add_torrent удаляется вместе со всем содержимым.
